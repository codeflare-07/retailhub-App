# Install RetailHub on Windows

## Build installer

```bash
npm install
npm run build
```

Output: `dist\RetailHub Setup 1.1.0.exe`

## Data folder

Your shop data is stored at:

`%APPDATA%\retailhub\`

Example:

`C:\Users\<YourName>\AppData\Roaming\retailhub\`

Data from the older **Guru Nanak Communication** install is copied automatically on first run.

## Default login

- **Login ID:** `admin`
- **Password:** `admin123`

Change under **Settings → Login ID & Password**.
